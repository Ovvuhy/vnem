# vnem Brand Assets

This folder contains the public vnem logo and banner used by the repository README and project surfaces.

- `logo.png`: transparent logo used in the README header.
- `banner.png`: README banner with project positioning copy.
- `banner-artwork.jpg`: text-free banner artwork for other surfaces.

These files are project identity artwork. They are not registry metadata and are not released under CC0 unless a maintainer explicitly says otherwise.
