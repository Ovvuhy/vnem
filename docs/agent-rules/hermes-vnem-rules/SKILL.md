# VNEM Agent Rules

Use this optional Hermes skill only when working on VNEM.

Start by opening the VNEM repo as the current working directory, then read the
root `AGENTS.md`. Follow the VNEM Agent Operating Rules, especially the Real
Improvement Doctrine, Anti-Placebo Law, Adaptive Goal Understanding, Research As
Understanding, Safety Boundaries, and Git / Commit / Push Rules.

Do not treat this skill as a replacement for the repo root `AGENTS.md`. The repo
file is the source of truth for VNEM project rules.

This artifact is intentionally repo-local. Do not install it globally unless the
user explicitly asks for that persistent Hermes setup.
