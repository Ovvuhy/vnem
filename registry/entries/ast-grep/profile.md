# ast-grep

ast-grep is a Rust and tree-sitter based CLI for structural code search, linting, and rewriting, letting agents match syntax patterns instead of fragile text snippets across many languages.

## Best For

- Agents that need precise code search or reviewable codemods across supported languages

## Recommended When

- Use when text grep would overmatch or miss syntax-equivalent code shapes.

## Review Notes

Trust tier: promising. Review status: manual-reviewed. Permissions: filesystem. Risk flags: reads-local-files, can-rewrite-code.
