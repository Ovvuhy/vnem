# Browserbase MCP

Browserbase MCP gives agents cloud browser sessions for navigation, extraction, QA, and web automation through MCP-compatible clients.

## Best For

- Hosted browser workflows where local Playwright sessions are insufficient

## Recommended When

- Use when browser automation needs cloud sessions, screenshots, or managed isolation.

## Review Notes

Trust tier: promising. Review status: manual-reviewed. Permissions: browser, network. Risk flags: browser-access, hosted-service, can-access-authenticated-pages.
