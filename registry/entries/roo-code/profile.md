# Roo Code

Roo Code is a VS Code-family coding agent that supports Code, Architect, Ask, Debug, and Custom Modes, MCP servers, checkpoints, codebase indexing, and model/provider configuration.

## Best For

- Teams that need explicit mode separation and model/provider flexibility inside VS Code-like editors

## Recommended When

- Use after confirming the current maintenance path and reviewing mode permissions for the project.

## Review Notes

Trust tier: promising. Review status: manual-reviewed. Permissions: filesystem, repository, shell, browser, network. Risk flags: can-edit-files, can-run-shell-commands, browser-access, community-handoff.
