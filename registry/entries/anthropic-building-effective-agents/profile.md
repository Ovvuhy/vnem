# Anthropic Building Effective Agents

Anthropic's Building Effective Agents guide distills practical patterns: start simple, compose workflows before agents, and use agentic autonomy only when feedback loops justify it.

## Best For

- Agent architecture decisions before adding framework complexity

## Recommended When

- Use before adding orchestration, swarms, or autonomy to decide whether the simpler pattern is enough.
