# Agent2Agent Protocol

Agent2Agent Protocol is a Google-backed open protocol for agent interoperability, enabling agents and agent runtimes to communicate across vendors and systems.

## Best For

- Architectures where agents from different systems need to exchange tasks or state

## Recommended When

- Use when multi-agent interoperability matters beyond one framework or vendor.
