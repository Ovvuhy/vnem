# DSPy

DSPy is a framework for programming and optimizing language-model pipelines with declarative modules, signatures, teleprompters, and evaluation-driven improvement.

## Best For

- Teams with datasets and metrics for optimizing language-model behavior

## Recommended When

- Use when repeated prompt tuning should be replaced by evaluation-driven optimization.
