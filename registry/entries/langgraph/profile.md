# LangGraph

LangGraph is a graph-based orchestration framework for stateful agents, durable execution, human-in-the-loop workflows, persistence, and multi-agent patterns.

## Best For

- Production agents that need explicit state machines, checkpoints, and human review

## Recommended When

- Use when agent behavior should be represented as a graph instead of hidden prompt flow.
