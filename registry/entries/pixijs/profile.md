# PixiJS

PixiJS is a lightweight 2D rendering library for rich browser graphics and interactive scenes. It is best treated as a renderer rather than a complete game framework: useful when the game needs fast sprites, filters, textures, and interaction, but the agent should keep rules and state architecture explicit.

## Best For

- 2D experiences where rendering performance matters more than engine-level game structure

## Recommended When

- Use when an agent needs a fast browser 2D renderer while keeping game rules and architecture custom.
