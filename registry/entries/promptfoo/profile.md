# promptfoo

promptfoo is an open-source tool for testing prompts, models, RAG systems, agents, and red-team scenarios with assertions and repeatable eval suites.

## Best For

- CI-friendly evals and prompt regression suites

## Recommended When

- Use when prompt and agent changes need fast pass/fail checks in CI.

Trust tier: promising. Review status: manual-reviewed. Permissions: network. Risk flags: can-send-test-data-to-model-providers.
