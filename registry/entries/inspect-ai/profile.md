# Inspect AI

Inspect AI is an open-source framework from the UK AI Safety Institute for large language model evaluations with tasks, solvers, scorers, and logs.

## Best For

- Teams that need structured evals and audit logs

## Recommended When

- Use when agent changes need evidence from repeatable eval tasks.

## Review Notes

Trust tier: promising. Review status: manual-reviewed. Permissions: network. Risk flags: none recorded.
