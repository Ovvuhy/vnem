# Custom Modes Roo Code

Custom Modes Roo Code is a community collection of Roo Code custom-mode and agent configurations with schemas, categorized agents, and supporting scripts for specialized mode-based workflows.

## Best For

- Teams experimenting with specialist mode design and willing to audit each mode before use

## Recommended When

- Use as an inspiration catalog; import only reviewed modes with scoped permissions.

## Review Notes

Trust tier: unreviewed. Review status: manual-reviewed. Permissions: filesystem, shell, browser, network. Risk flags: community-maintained, can-grant-broad-tool-access, mode-audit-required.
