# Swarms

Swarms is a Python multi-agent orchestration framework with agent primitives, sequential, concurrent, graph, hierarchical, mixture-of-agents, and router-style workflows for production agent systems.

## Best For

- Python teams evaluating explicit multi-agent orchestration patterns

## Recommended When

- Use when the team needs to compare several explicit multi-agent execution patterns in Python.

## Review Notes

Trust tier: promising. Review status: manual-reviewed. Permissions: network, filesystem. Risk flags: multi-agent-cost-amplification, model-api-required, tool-side-effects.
