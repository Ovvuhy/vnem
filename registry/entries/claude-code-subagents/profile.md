# Claude Code Subagents

Claude Code subagents are specialized agent definitions with focused prompts, tool permissions, and separate context windows for delegated coding and research tasks.

## Best For

- Repeated specialist tasks like test writing, migration review, docs cleanup, and codebase exploration

## Recommended When

- Use when a recurring specialist role can be given clear instructions and a bounded tool set.

## Review Notes

Trust tier: verified. Review status: manual-reviewed. Permissions: filesystem, repository, shell, network. Risk flags: license-not-asserted, can-edit-files, can-run-shell-commands.
