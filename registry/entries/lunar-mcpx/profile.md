# Lunar MCPX

Lunar MCPX is an MCP gateway that sits between MCP clients and backend MCP servers, providing a single endpoint, aggregated tool catalog, dynamic routing, access rules, authentication, and audit-oriented enterprise controls.

## Best For

- Teams that have outgrown direct per-client MCP server configuration

## Recommended When

- Use when direct MCP configuration creates too much tool-schema, auth, or audit complexity.

## Review Notes

Trust tier: promising. Review status: manual-reviewed. Permissions: network. Risk flags: gateway-operational-risk, centralizes-tool-access, requires-policy-review.
