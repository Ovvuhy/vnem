# ripgrep

ripgrep is a mature Rust command-line search tool that recursively searches files for regex patterns while respecting gitignore rules, making it a strong default grep replacement for coding agents inspecting repositories.

## Best For

- Coding agents that need fast lexical search before reading files

## Recommended When

- Use as the default text-search primitive before opening files or proposing code changes.
