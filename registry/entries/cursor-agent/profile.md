# Cursor Agent

Cursor Agent is Cursor's editor-native coding agent for navigating codebases, applying multi-file edits, using project rules, running terminal-backed tasks, and working inside Cursor's IDE context.

## Best For

- Developers who want agentic edits inside Cursor's editor workflow

## Recommended When

- Use when Cursor project rules and editor context are central to the coding workflow.

## Review Notes

Trust tier: promising. Review status: manual-reviewed. Permissions: filesystem, repository, shell, network. Risk flags: license-not-asserted, can-edit-files, can-run-shell-commands, hosted-service.
