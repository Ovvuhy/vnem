# Cursor Agent

Cursor Agent is Cursor's editor-native coding agent for navigating codebases, applying multi-file edits, using project rules, and running terminal-backed tasks.

## Best For

- Developers who want agentic edits inside an editor workflow

## Recommended When

- Use when project rules and editor context are central to the coding workflow.

## Review Notes

Trust tier: promising. Review status: manual-reviewed. Permissions: filesystem, repository, shell, network. Risk flags: license-not-asserted, can-edit-files, can-run-shell-commands.
