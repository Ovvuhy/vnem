# Graphiti

Graphiti is an open-source framework for building temporal knowledge graphs that help agents retrieve facts, events, and relationships over time.

## Best For

- Agent memory that benefits from entities, relationships, and time-aware retrieval

## Recommended When

- Use when vector search alone is not enough for agent memory.
