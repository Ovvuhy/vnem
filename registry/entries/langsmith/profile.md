# LangSmith

LangSmith is LangChain's observability, tracing, evaluation, dataset, and monitoring platform for debugging and improving LLM and agent applications.

## Best For

- LangChain and LangGraph teams needing hosted traces and evals

## Recommended When

- Use when observability should be tightly integrated with LangChain or LangGraph.
