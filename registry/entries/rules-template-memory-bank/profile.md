# Rules Template Memory Bank

Rules Template is a community rule and memory-bank template for Cursor, Cline, Roo Code, and Windsurf that organizes agent instructions, workflow phases, and persistent project context.

## Best For

- Teams willing to maintain agent memory files and review rule changes like code

## Recommended When

- Use as a pattern source, then copy only the minimal ideas that match the repository's workflow.

## Review Notes

Trust tier: unreviewed. Review status: manual-reviewed. Permissions: filesystem. Risk flags: license-not-asserted, community-maintained, stale-memory-risk, can-overconstrain-agents.
