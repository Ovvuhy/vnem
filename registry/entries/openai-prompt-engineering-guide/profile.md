# OpenAI Prompt Engineering Guide

OpenAI's prompt engineering guide is an official source for improving instructions with clear goals, context, examples, output formats, and iterative evaluation.

## Best For

- Prompt rewrites where clarity, examples, constraints, and output shape determine quality

## Recommended When

- Use when a prompt is vague, underspecified, hard to evaluate, or missing an output contract.
