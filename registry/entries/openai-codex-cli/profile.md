# OpenAI Codex CLI

OpenAI Codex CLI is an open-source terminal coding agent for reading repositories, editing files, running commands, and producing reviewable changes with user-controlled execution.

## Best For

- Terminal-native coding work with reviewable diffs
- Repositories that need command execution under approval

## Recommended When

- Use when a project needs a local coding agent that can inspect files, edit code, and verify changes.

## Review Notes

Trust tier: promising. Review status: manual-reviewed. Permissions: filesystem, repository, shell, network. Risk flags: can-edit-files, can-run-shell-commands.
