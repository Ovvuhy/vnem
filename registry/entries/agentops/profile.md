# AgentOps

AgentOps is an observability platform for AI agents that records sessions, tool calls, costs, errors, and analytics across agent frameworks.

## Best For

- Agent-specific telemetry where session timelines and costs matter

## Recommended When

- Use when agent operations need cost, event, and failure visibility.

## Review Notes

Trust tier: promising. Review status: manual-reviewed. Permissions: network. Risk flags: trace-data-retention, hosted-service.
