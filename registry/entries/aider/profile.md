# Aider

Aider is an open-source AI pair-programming tool that edits local files, works with Git, maps repositories, and supports many language models from the terminal.

## Best For

- Small-to-medium coding tasks where Git diffs and local control matter

## Recommended When

- Use when a lightweight terminal pair programmer is enough and Git history is the review surface.

## Review Notes

Trust tier: promising. Review status: manual-reviewed. Permissions: filesystem, repository, network. Risk flags: can-edit-files.
