# CrewAI

CrewAI is a Python framework for creating role-based AI agent teams, tasks, crews, flows, knowledge, tools, and multi-agent process automation.

## Best For

- Clear task pipelines where agent roles and task ownership improve readability

## Recommended When

- Use when a crew/task mental model is clearer than a graph or raw SDK loop.
