# Supabase MCP

Supabase MCP is a community Supabase MCP server for project, database, SQL, and platform operations through MCP-compatible agent clients.

## Best For

- Supabase-backed apps where schema and project context improve agent work

## Recommended When

- Use when database scope can be limited and SQL changes are reviewed before execution.

## Review Notes

Trust tier: promising. Review status: manual-reviewed. Permissions: database, network. Risk flags: database-access, can-mutate-data, requires-token.
