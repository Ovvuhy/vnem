# Firecrawl MCP

Firecrawl MCP lets agents crawl, scrape, search, and extract web content through Firecrawl's web data APIs for research and ingestion workflows.

## Best For

- Research agents that need structured web extraction

## Recommended When

- Use when web extraction is permitted and citations or raw sources remain available.
