# ai.fincontext/fincontext

ai.fincontext/fincontext is an MCP server for agentic systems. Read-only bank & investment accounts via Plaid: balances, holdings, transactions, SQL analytics.

## When To Use

Use this when an agent needs extend agents with external tools through a tool surface that can be reviewed and permissioned.
