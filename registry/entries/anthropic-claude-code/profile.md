# Claude Code

Claude Code is Anthropic's terminal coding agent for codebase understanding, file edits, command execution, GitHub workflows, MCP tool use, memory files, and subagents.

## Best For

- Codebases that benefit from explicit project memory and specialist subagents

## Recommended When

- Use when Claude Code's memory files, MCP integration, and subagents map to the team's coding workflow.

## Review Notes

Trust tier: promising. Review status: manual-reviewed. Permissions: filesystem, repository, shell, network. Risk flags: license-not-asserted, can-edit-files, can-run-shell-commands.
