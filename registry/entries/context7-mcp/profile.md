# Context7 MCP

Context7 MCP is an MCP server that gives agents up-to-date library documentation and code examples so generated code can match current APIs.

## Best For

- Coding agents working with fast-moving frameworks and APIs

## Recommended When

- Use when an agent repeatedly needs current framework or library docs.
