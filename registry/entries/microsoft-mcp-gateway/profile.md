# Microsoft MCP Gateway

Microsoft MCP Gateway is a reverse proxy and management layer for MCP servers focused on scalable, session-aware, stateful routing and lifecycle management in Kubernetes and Azure-oriented environments.

## Best For

- Teams already operating Azure or Kubernetes infrastructure for agent tooling

## Recommended When

- Use when MCP routing must be governed in a cloud-native environment with explicit identity and lifecycle controls.
