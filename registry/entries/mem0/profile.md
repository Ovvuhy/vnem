# Mem0

Mem0 is an agent memory layer for extracting, storing, searching, and updating long-term user or application memory across AI assistants and agent workflows.

## Best For

- Assistants that need personalization over repeated sessions

## Recommended When

- Use when memory improves repeated work and users can inspect or delete stored memories.
