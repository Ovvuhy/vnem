# Ragas

Ragas is an evaluation framework for RAG and LLM applications with metrics for retrieval, generation quality, testset generation, and experiment workflows.

## Best For

- Knowledge assistants and retrieval-heavy agent systems

## Recommended When

- Use when the agent depends on retrieval quality and grounded answers.
