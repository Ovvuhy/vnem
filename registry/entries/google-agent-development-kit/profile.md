# Google Agent Development Kit

Google Agent Development Kit is an open-source framework for building, evaluating, and deploying agents with tools, callbacks, memory, and multi-agent patterns.

## Best For

- Google Cloud and Gemini-native agent applications

## Recommended When

- Use when deployment and evaluation should stay close to Google's agent ecosystem.
