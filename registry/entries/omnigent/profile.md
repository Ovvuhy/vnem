# Omnigent

Omnigent is an open-source AI agent meta-harness that orchestrates Claude Code, Codex, Cursor, and custom agents under a unified policy and sandboxing layer, with real-time multi-device collaboration.

## Best For

- Teams composing multi-agent workflows with policy and sandbox requirements
- Projects that need to swap agent harnesses without rewriting workflows

## Recommended When

- Use when a project needs to orchestrate multiple coding agents with unified policy, sandboxing, and cross-harness coordination.

Trust tier: promising. Review status: bot-reviewed. Permissions: filesystem, repository, shell, network. Risk flags: can-orchestrate-agents, can-enforce-policies, network-access.
