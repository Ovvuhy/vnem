# OpenHands

OpenHands is an open-source software engineering agent platform for code editing, terminal use, browsing, issue work, and reproducible development environments.

## Best For

- Sandboxed SWE-agent style tasks that need a full environment

## Recommended When

- Use when tasks need an interactive development environment rather than only patch generation.

## Review Notes

Trust tier: promising. Review status: manual-reviewed. Permissions: filesystem, repository, shell, browser, network. Risk flags: can-edit-files, can-run-shell-commands, browser-access.
