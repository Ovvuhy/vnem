# Zep

Zep is a memory platform for AI agents that builds temporal knowledge graphs and retrieves relevant user, session, and business context for agent applications.

## Best For

- Hosted memory systems that need graph-shaped context

## Recommended When

- Use when temporal graph memory is worth the operational and privacy overhead.

## Review Notes

Trust tier: promising. Review status: manual-reviewed. Permissions: network, database. Risk flags: license-not-asserted, stores-user-memory, hosted-service.
