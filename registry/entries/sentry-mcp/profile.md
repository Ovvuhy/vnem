# Sentry MCP

Sentry MCP gives agents access to Sentry errors, issues, traces, and debugging context so coding agents can investigate production failures with source-backed evidence.

## Best For

- Debugging production issues with trace and error context

## Recommended When

- Use when a coding agent needs error evidence before proposing a fix.

## Review Notes

Trust tier: promising. Review status: manual-reviewed. Permissions: network. Risk flags: requires-token, may-expose-user-data.
