# Pydantic AI

Pydantic AI is a Python agent framework that emphasizes typed dependencies, structured outputs, model-agnostic providers, tools, and testable agent logic.

## Best For

- Python apps already using Pydantic and typed service boundaries

## Recommended When

- Use when agent outputs should be validated like normal application data.
