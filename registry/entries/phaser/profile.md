# Phaser

Phaser is an open-source HTML5 framework for browser-based 2D games. It is a good default when an agent needs more than a raw Canvas loop: scenes, sprites, asset loading, input, cameras, animation, audio, and Canvas/WebGL rendering.

## Best For

- 2D games that need a mature game framework instead of custom Canvas code

## Recommended When

- Use when a browser game needs scenes, sprites, asset loading, collisions, cameras, audio, and a tested 2D engine.

## Review Notes

Trust tier: promising. Review status: manual-reviewed. Permissions: browser, canvas, webgl, audio. Risk flags: browser-runtime, canvas-webgl-rendering.
