# ugrep

ugrep is a high-performance grep-compatible search utility with interactive TUI, Boolean and fuzzy search, archive and compressed-file search, document search, and structured output options.

## Best For

- Repository audits that need search beyond source text files

## Recommended When

- Use when search needs include archives, compressed files, documents, or Boolean/fuzzy matching.
