# Stripe Agent Toolkit

Stripe Agent Toolkit provides agent-facing Stripe tools, including MCP support, so agents can interact with Stripe APIs under explicit payment and commerce controls.

## Best For

- Agent commerce flows where Stripe is the payment system of record

## Recommended When

- Use only when payment authorization, receipts, and test-mode verification are explicit.

## Review Notes

Trust tier: promising. Review status: manual-reviewed. Permissions: payments, network. Risk flags: payments, requires-secret, can-affect-money.
