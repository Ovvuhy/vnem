# Excalibur.js

Excalibur.js is a TypeScript-first 2D HTML5 canvas game engine. It is a good fit when an agent needs actors, scenes, collision, input, assets, and a structured loop while staying in a browser-native TypeScript workflow.

## Best For

- TypeScript-oriented 2D games that need engine structure without moving into 3D

## Recommended When

- Use when a 2D browser game should stay TypeScript-first and benefit from actors, scenes, collision, and asset structure.

## Review Notes

Trust tier: promising. Review status: manual-reviewed. Permissions: browser, canvas, audio. Risk flags: browser-runtime, pre-1.0-api.
