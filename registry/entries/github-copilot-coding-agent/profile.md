# GitHub Copilot Coding Agent

GitHub Copilot Coding Agent is a cloud coding agent that can be assigned GitHub issues, work in a secure environment, and open pull requests for review.

## Best For

- GitHub-native teams that want issue-to-PR workflows with existing branch protections

## Recommended When

- Use when the repository already relies on GitHub Issues, protected branches, and PR review workflows.

## Review Notes

Trust tier: promising. Review status: manual-reviewed. Permissions: repository, network. Risk flags: license-not-asserted, mutates-repositories, hosted-service.
