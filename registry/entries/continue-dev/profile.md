# Continue

Continue is an open-source AI code assistant and agent platform for VS Code and JetBrains with configurable models, context providers, and team rules.

## Best For

- Teams that want configurable editor AI instead of a single hosted assistant

## Recommended When

- Use when model choice, local context configuration, and editor integration are important.

## Review Notes

Trust tier: promising. Review status: manual-reviewed. Permissions: filesystem, repository, network. Risk flags: can-edit-files.
