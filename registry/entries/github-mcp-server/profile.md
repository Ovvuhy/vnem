# GitHub MCP Server

GitHub MCP Server is GitHub's official MCP server for repository, issue, pull request, search, workflow, and code operations through agent clients.

## Best For

- Repository workflows where GitHub is the system of record

## Recommended When

- Use when an agent needs GitHub context or PR workflows and token scope can be constrained.

## Review Notes

Trust tier: promising. Review status: manual-reviewed. Permissions: repository, network. Risk flags: mutates-repositories, requires-token.
