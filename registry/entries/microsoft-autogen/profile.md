# Microsoft AutoGen

Microsoft AutoGen is an agent framework for building event-driven, distributed, and multi-agent applications with AgentChat abstractions and extension points.

## Best For

- Research and product teams exploring conversational multi-agent architectures

## Recommended When

- Use when agents need to communicate through structured conversations and events.
