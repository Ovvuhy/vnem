# Semantic Kernel

Microsoft Semantic Kernel is an SDK for integrating AI services, prompts, tools, planners, and agents into C#, Python, and Java applications.

## Best For

- Teams building AI features inside existing Microsoft or .NET-oriented applications

## Recommended When

- Use when app integration matters more than standalone agent orchestration.
