# Vercel AI SDK

Vercel AI SDK is a TypeScript toolkit for model calls, streaming UI, tool calling, structured output, and agent-style application workflows.

## Best For

- Next.js and frontend-heavy AI products

## Recommended When

- Use when the agent surface is a web application rather than a backend-only worker.
