# Mastra

Mastra is a TypeScript agent framework for agents, workflows, RAG, memory, MCP integration, evals, and deployment in JavaScript applications.

## Best For

- JavaScript and TypeScript teams building production agent apps

## Recommended When

- Use when Node-native agent infrastructure is preferred over Python frameworks.
