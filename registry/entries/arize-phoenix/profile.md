# Arize Phoenix

Arize Phoenix is an open-source AI observability and evaluation platform for tracing, debugging, and evaluating LLM and agent applications.

## Best For

- Teams that want open observability with OpenTelemetry-friendly traces

## Recommended When

- Use when debugging agent behavior requires traces, datasets, and eval feedback.
