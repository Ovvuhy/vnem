# Babylon.js

Babylon.js is a full-featured 3D browser engine for WebGL/WebGPU games and interactive applications. It is heavier than a focused renderer, but gives agents engine-level primitives for scenes, materials, assets, GUI, XR, and physics-oriented workflows.

## Best For

- 3D browser games that benefit from a full engine instead of a focused renderer

## Recommended When

- Use when an agent needs a full 3D web engine with browser verification, not just a graphics library.

## Review Notes

Trust tier: promising. Review status: manual-reviewed. Permissions: browser, canvas, webgl, webgpu, audio. Risk flags: browser-runtime, gpu-rendering.
