# Letta

Letta is an open-source framework for stateful agents with memory, tools, archival storage, and server APIs descended from the MemGPT research line.

## Best For

- Long-lived agents that need inspectable memory state

## Recommended When

- Use when memory should be first-class and inspectable rather than hidden in prompts.

## Review Notes

Trust tier: promising. Review status: manual-reviewed. Permissions: network, database. Risk flags: stores-user-memory, data-retention-risk.
