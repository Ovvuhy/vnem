# Three.js

Three.js is a JavaScript 3D graphics library for custom browser scenes. It is useful for games and interactive views where 3D is central, but it is not a full game framework by itself; UI, physics, assets, and state still need deliberate choices.

## Best For

- Custom 3D browser experiences where a focused graphics library is preferable to a full game engine

## Recommended When

- Use when the core game mechanic or visual direction is actually 3D and the agent can verify the rendered scene in a browser.

## Review Notes

Trust tier: promising. Review status: manual-reviewed. Permissions: browser, canvas, webgl, webgpu. Risk flags: browser-runtime, gpu-rendering.
