# OpenAI Agents SDK

OpenAI Agents SDK is a lightweight framework for agent loops, tools, handoffs, guardrails, tracing, and multi-agent workflows built around OpenAI models.

## Best For

- OpenAI-native Python agent applications with handoffs and tracing

## Recommended When

- Use when the team wants a compact official SDK instead of a larger graph runtime.
