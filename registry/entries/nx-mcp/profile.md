# nx-mcp

nx-mcp is a Model Context Protocol server for Nx monorepos. It exposes Nx project graph, task execution, and code generation tools to MCP-compatible AI agents.

## Best For

- Teams using Nx monorepos who want AI agent integration
- Agents that need to understand project dependencies and run tasks

## Recommended When

- Use when working in an Nx monorepo and want AI agents to understand project structure and run tasks
