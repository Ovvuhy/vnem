# Matter.js

Matter.js is a JavaScript 2D rigid-body physics engine for browser games and simulations. It is useful when custom overlap checks are not enough, but the game does not need a heavier physics or 3D stack.

## Best For

- 2D browser games that need approachable rigid-body physics

## Recommended When

- Use when game rules need 2D rigid-body behavior and the agent can test collision outcomes in browser gameplay.
