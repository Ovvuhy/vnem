# gptme

gptme is a terminal-based AI agent equipped with local tools: writes code, uses the terminal, browses the web. Supports multiple LLM backends and can be extended into a persistent autonomous agent.

## Best For

- Developers who want a terminal-native agent with local tool access
- Persistent autonomous agents with custom memory

## Recommended When

- Use when you want a terminal-native agent with local tool execution and web browsing
