# Clude

Clude is an early-stage cognitive memory layer and MCP server for AI agents, with hosted, self-hosted, and local modes for storing, recalling, decaying, consolidating, and exporting agent memories.

## Best For

- Agent products that need inspectable memory and can define retention boundaries

## Recommended When

- Use when agent memory should be explicit, searchable, and portable across MCP-compatible clients.

## Review Notes

Trust tier: promising. Review status: manual-reviewed. Permissions: network, filesystem, database. Risk flags: stores-user-memory, data-retention-risk, early-stage, hosted-memory-service.
