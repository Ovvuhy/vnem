# LlamaIndex Agents

LlamaIndex Agents provide tool-using and workflow-oriented agent patterns for retrieval-heavy applications, knowledge systems, document search, and RAG pipelines.

## Best For

- RAG-heavy assistants and knowledge workflows

## Recommended When

- Use when retrieval and document indexing are the center of the agent design.
