# Model Context Protocol

Model Context Protocol is an open protocol and schema for connecting AI applications to external tools, resources, prompts, and contextual data through standardized MCP servers.

## Best For

- Teams that need portable, reviewable tool connections across agent clients

## Recommended When

- Use when agent tools should be discoverable, permissioned, and reusable across clients.
- Review server permissions and transport trust boundaries before connecting high-risk systems.

## Review Notes

Trust tier: verified. Review status: manual-reviewed. Permissions: none recorded. Risk flags: requires-server-review, tool-permissions-are-server-defined.
