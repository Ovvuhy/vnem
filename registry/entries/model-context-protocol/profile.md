# Model Context Protocol

Model Context Protocol is an open protocol for connecting AI applications to external tools, resources, prompts, and contextual data through standardized servers.

## Best For

- Teams that need portable tool connections across agent clients

## Recommended When

- Use when agent tools should be discoverable, permissioned, and reusable across clients.
