# PlayCanvas Engine

PlayCanvas Engine is an open-source browser 3D game engine built around WebGL2 and WebGPU. It is useful when an agent needs a browser-first 3D engine with rendering, input, assets, animation, audio, physics integration, and WebXR pathways.

## Best For

- 3D browser games or visualizations that benefit from an engine with a browser-based ecosystem

## Recommended When

- Use when a project needs a browser-first 3D game engine with rendering, input, assets, audio, animation, and XR paths.
