# Playwright MCP

Playwright MCP is Microsoft's browser automation MCP server for giving agents structured access to pages, snapshots, actions, and web UI testing flows.

## Best For

- Frontend QA, local app inspection, and browser-based debugging

## Recommended When

- Use when a coding agent needs real browser evidence instead of guessing from code.

## Review Notes

Trust tier: promising. Review status: manual-reviewed. Permissions: browser, network, filesystem. Risk flags: browser-access, can-access-authenticated-pages.
