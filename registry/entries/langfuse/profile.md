# Langfuse

Langfuse is an open-source LLM engineering platform for tracing, prompt management, evaluation, metrics, datasets, and observability across agent systems.

## Best For

- Teams that want open-source tracing and prompt/eval operations

## Recommended When

- Use when prompt versions, traces, and eval datasets should live together.

## Review Notes

Trust tier: promising. Review status: manual-reviewed. Permissions: network. Risk flags: trace-data-retention.
