# KAPLAY

KAPLAY is a JavaScript and TypeScript 2D game library for fast browser-game prototyping. It is most useful when an agent needs a small playable game quickly, with simple game objects, collisions, input, sprites, and concise examples.

## Best For

- Fast 2D browser-game prototypes where concise game code matters most

## Recommended When

- Use when an agent should make a small 2D browser game feel playable quickly without a heavy engine.
